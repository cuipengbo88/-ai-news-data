# AI 日报 2026-06-11

## 🔥 重点

### 1. Skill 生态爆发：Agent 不是聊天框，Skill 是关键中间层

今天多条消息指向同一个趋势：**Skill/插件生态正在成为 AI Agent 的核心战场**。

歸藏（guizang.ai）发出万字长文，核心结论一针见血——Agent 不会抹平能力差距，反而会把差距放大数百倍。头部用户已经搭建了文档、规则、memory、MCP、CLI、权限管理的完整体系，而普通用户还停在聊天框。Skill 是普通用户用好 Agent 的"关键中间层"——好的 Skill 需要设计、维护、分发，生态不能只做仓库列表，需要内容、产品、案例、反馈形成飞轮。

同一天，小互开源了一个公众号排版 Skill：一句话完成排版、封面生成、一键发送到公众号草稿箱，20 种主题颜色可选，自动分析内容进行排版。安装即用，全程无需手动操作。向阳乔木也发布了 qiaomu-ai-prd——面向 AI Agent 的 PRD 生成 Skill，先让 AI 生成结构化 PRD，再交给 AI 开发，功能完整度翻倍。

**我的判断**：Skill 生态正在经历"从工具到方法论"的跃迁。这跟你正在做的 AI 热点总结 Skill 是同一波浪潮——把"自己会用 AI"变成"别人能通过你的 Skill 用好 AI"。对于编导来说，这意味着你积累的脚本改写、文案生成、数据分析经验，可以被封装成可复用的 Skill，从卖服务变成卖方法论。

### 2. EurekAgent + MaxProof：AI 自主科研的转折点来了

今天两个重磅论文/项目指向同一个方向：**AI 开始自主做科学发现**。

**EurekAgent** 提出了"环境工程化"思路——不靠更聪明的模型，而是设计一个让 AI 能自主探索的执行环境（权限隔离、产物管理、成本感知、人在回路四层工程）。结果是：以不到 **11 美元**的总 API 成本，发现了新的 26 圆填充结果，在数学、内核工程、机器学习任务上均取得新 SOTA。

**MaxProof**（MiniMax-M3）则从另一个路线突破：用群体级别测试时扩展框架，让单个模型同时扮演生成器、验证器、精炼器和排序器，在候选证明群体中搜索最优解。结果是：IMO 2025 达 35/42，USAMO 2026 达 36/42，**均超过人类金牌阈值**。

**为什么放在一起看**：EurekAgent 走的是"低成本、环境驱动"路线，MaxProof 走的是"群体博弈、验证闭环"路线——两条路都跑通了。这意味着 AI 自主科研的范式已经不只是"大模型 + 更多算力"，而是系统设计的竞争。

**我的判断**：对你做 vibe coding 有直接启发——EurekAgent 的"环境工程化"思路完全可以降维用到产品开发中：设计好 Agent 的沙箱环境、版本管理、成本监控、回滚机制，比选什么模型更重要。

## 🎬 编导可用的

### 小互的公众号排版 Skill（今日最佳实用工具）
一句话完成公众号排版、封面生成并一键发送到草稿箱。20 种主题颜色，自动分析内容进行排版，不是简单的 Markdown 渲染。**为什么值得关注**：如果你服务的实体老板有公众号运营需求，这个工具能省掉排版设计师的钱——从"写好文章 → 导出 → 手动排版 → 做封面 → 上传"变成"一句话搞定"。全流程从 30-60 分钟缩到 1 分钟。已开源，可直接安装。

### iOS 27 视觉智能营养识别
iPhone 相机对准食物，自动识别加工程度、蛋白质、含糖量，给出营养价值评级。虽然没有精确卡路里，但"深度加工食品"标记对健康内容创作很有用。**为什么值得关注**：餐饮/食品类实体老板的内容素材——"用 iOS 27 扫一下你家后厨的食材"比"我们食材很好"有说服力得多。这个功能可能在 9 月正式发布后成为短视频爆款素材。

### PixVerse AI 叙事创作
有创作者用 PixVerse 做出了 24 集的连载动画故事，情感线完整、角色有成长弧。**为什么值得关注**：AI 连续剧创作正在从"单集炫技"进化到"长篇叙事"。对做餐饮、美业、教育等实体老板的编导来说，可以考虑用 AI 给品牌做连载短剧——成本远低于真人拍摄，但需要编剧思维。

## 🛠️ Vibe Coding 可玩的

### qiaomu-ai-prd（AI 专用 PRD 生成 Skill）
向阳乔木发布的 AI 专用 PRD 生成 Prompt，已打包为 Skill。先用它生成结构化 PRD 文档，再交给 AI Agent 开发，功能完整度远超人工手写需求。Fable 5 的在线 Photoshop 项目正是用这个方法开发的。**上手难度**：极低（`npx skills add joeseesun/qiaomu-ai-prd` 一键安装）。**适用场景**：当你有一个产品想法但不知道怎么写清楚让 AI 理解时，这是目前最好的"翻译器"。

### FlashMemory（DeepSeek-V4 显存压缩）
将 DeepSeek-V4 的 1M 上下文显存从 10GB 压到 1.3GB，且长文本准确率反而提升。训练不需要加载基座模型，成本很低。**上手难度**：中（需要部署）。**适用场景**：如果你在本地跑 DeepSeek-V4 做 vibe coding，显存不再是大问题。尤其是处理大型代码库时，1.3GB 显存意味着消费级显卡就能吃下整个项目。
